Truth Finder

Convert a logic table into code. For people, like me, who find it hard to visualise multiple inputs and map them to an output this can be very helpful. Due to the recursive nature of the size of a Logic Table up to 4 variables are supported maximum. Hopefully this spaghetti code can help you reduce your spaghetti code :)
